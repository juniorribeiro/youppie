# Excluir Quiz

## O que você pode fazer

Excluir um quiz remove-o permanentemente do sistema, incluindo todos os dados associados (leads, respostas, analytics). Use com cuidado.

## Como funciona

### Passo 1: Acesse o quiz no dashboard

No Dashboard, encontre o quiz que você quer excluir.

### Passo 2: Clique no ícone de lixeira

Passe o mouse sobre o card do quiz. Um ícone de lixeira aparecerá. Clique nele.

### Passo 3: Confirme a exclusão

Uma mensagem de confirmação aparecerá perguntando se você tem certeza. Digite o nome do quiz ou confirme a exclusão.

> **⚠️ Atenção:** A exclusão é permanente e não pode ser desfeita. Todos os dados serão perdidos.
