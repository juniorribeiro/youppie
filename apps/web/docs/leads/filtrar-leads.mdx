# Filtrar Leads

## O que você pode fazer

Os filtros permitem que você encontre leads específicos rapidamente. Você pode filtrar por quiz, data, ou fazer uma busca por texto.

## Como funciona

### Passo 1: Acesse os filtros

Na página de Leads, após selecionar um quiz, você verá uma seção de **filtros** no topo.

### Passo 2: Aplique os filtros

Você pode usar:
- **Quiz**: Selecione um quiz específico
- **Data inicial**: Defina uma data de início
- **Data final**: Defina uma data de término
- **Busca**: Digite texto para buscar em nome, e-mail ou telefone

### Passo 3: Visualize os resultados

Os leads são filtrados automaticamente conforme você aplica os filtros.
